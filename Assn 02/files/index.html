This is the index page.

List of Commands:
  • mnm-meme: Download a meme image
  • mnm-text: Download the story of my life
  • info: Displays an information page
  • link: Displays a link to a download (incorporates /download)
  • page1: Displays the page 'page1'
  • echo: Legacy command
  • exit: Terminates the server
If you enter a command that isn't recognized, then you will surely get an error.
